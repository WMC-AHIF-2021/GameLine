Overview



Why did we start this projekt? We decided to do this, because we wanted to see, how the early games looked liked. Its also incredible how it all improved so much, compared to the games now. And GameLine is a website for the early games. We show you how they looked like and what the task were. We also created our own game, which is linked at the top of the page. We also described 3 games, which were Ant Attack, Wolfensteiner3d and Unreal Tournament. There are also a lot of more games, we want to show you in the future. The Website itself is structured with a linked description, which are on top of the page and also the description of the programmer of the Webistes. Micheal and Samuel , we 2 created this Website. Our Project



Our Project idea was to programm a little game, where you can jump, go right, left and shoot at enemies. To Programm this Game was at first very tricky, because we had no experience with 3d modells so we needed to download the templetes for the 3d modells. Then we needed to programm the elements for walking and shooting and how the enemies got hit and so on. We needed the entire easter holidays for the Game, but in the end it worked out. It looked great for our first game. The Link for the game is on the top of the main page.
