Our Project



Our Project idea was to programm a little game, where you can jump, go right, left and shoot at enemies. To Programm this Game was at first very tricky, because we had no experience with 3d modells so we needed to download the templetes for the 3d modells. Then we needed to programm the elements for walking and shooting and how the enemies got hit and so on. We needed the entire easter holidays for the Game, but in the end it worked out. It looked great for our first game. The Link for the game is on the top of the main page.
